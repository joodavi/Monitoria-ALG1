30/03 Comandos Condicionais

O IF é um comando condicional que levará o programa a tomar uma decisão baseado em uma condição estabelecida

Se a condição tiver valor TRUE, o progrma executará o comando do bloco IF. Caso contrário, segurá para a próxima instrução

06/04 Laços de Repetição

O WHILE é um laço de repetição que utiliza o princípio do contador. Através do WHILE, podemos executar o código até que determinada condição seja satisfeita

O FOR é um laço de repetição que é utilizado principalmente para percorrer um vetor, pegando o valor de cada um de seus elementos. Para cada um desses elementos ele irá realizar um bloco de comando

13/04 Listas

Uma LISTA é um conjunto ordenado de valores, em que cada valor é identificado por um índice. Os valores que compõem uma lista são chamados de elementos
